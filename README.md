# PTTK-ThuatToan-OnTap — 4 chương tương tác

Tài liệu ôn tập **Phân Tích Thiết Kế Thuật Toán** với 4 chương trọng tâm thi, bám sát giáo trình Th.S. Nguyễn Văn Linh - Đại học Cần Thơ 2003.

## 🎯 Mục tiêu

Dành cho sinh viên "mất gốc" vẫn muốn đạt 10 điểm. Mỗi bài được thiết kế:
- **Lý thuyết + Ý tưởng** trước, mã sau.
- **Giả mã (Pascal/C) + Cài đặt Python** song song.
- **Mô phỏng từng bước** — bấm "Bước tiếp" xem thuật toán chạy.
- **Bài tập tự giải** có đáp án chi tiết 5 phần (công thức + cách áp dụng + tính + kết quả + kiểm tra).
- **Quiz** kiểm tra hiểu bài.
- **Tư duy thuật toán** — chìa khóa giải bài lạ trong thi.
- **Cách trình bày bài thi** chuẩn để được điểm tối đa.

## 📖 Cách học hiệu quả

1. Đọc lý thuyết + ý tưởng trước — phải hiểu vì sao.
2. Chạy mô phỏng từng bước — bấm "Bước tiếp".
3. Đọc giả mã + Python song song.
4. Làm bài tập tự sinh — nhập dữ liệu của bạn.
5. Làm quiz để kiểm tra.
6. Đọc "Tư duy thuật toán" cuối mỗi bài — để giải bài lạ.

## 🗓️ Lộ trình 3 buổi

- **Buổi 1**: Nền tảng (BigO + 3 thuật toán sắp xếp đơn giản)
- **Buổi 2**: BigO nâng cao + QuickSort + HeapSort
- **Buổi 3**: Greedy + DP
